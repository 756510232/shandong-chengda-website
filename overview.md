# 山东诚达信息科技有限公司 - 企业官网开发完成

## 项目概述
为山东诚达信息科技有限公司开发了企业官网，主营业务车抵贷（车辆抵押贷款），包含4个页面，风格简约大气。

## 页面清单
| 页面 | 文件 | 内容 |
|------|------|------|
| 首页 | `index.html` | Hero区、数据统计、业务优势(6项)、服务流程(4步)、CTA |
| 关于我们 | `about.html` | 公司简介、发展历程时间线(5个节点)、企业价值观(4项)、资质保障 |
| 产品服务 | `services.html` | 3款车抵贷产品、申请条件、办理流程、还款方式、6条FAQ |
| 联系我们 | `contact.html` | 联系表单(含验证)、联系信息、地图区域、服务网点 |

## 技术架构
- **纯静态网站**：HTML + CSS + JavaScript，无需后端，部署简单
- **共享样式系统**：`css/style.css` — 完整设计令牌系统
- **交互脚本**：`js/main.js` — 主题切换/滚动动画/表单验证/FAQ手风琴

## 设计亮点
- **配色方案**：深蓝(#0f2c4c) + 鎏金(#c9a961)，金融质感
- **明暗主题切换**：支持亮色/暗色/跟随系统，localStorage持久化
- **滚动入场动画**：IntersectionObserver 实现，带延迟错落效果
- **数字滚动动画**：数据统计区数字从0滚动到目标值
- **磁吸按钮效果**：鼠标移动时按钮微跟随
- **响应式设计**：桌面/平板/手机三端适配
- **玻璃拟态**：部分卡片使用 backdrop-filter 毛玻璃效果
- **表单验证**：手机号正则验证、邮箱格式验证、必填项验证
